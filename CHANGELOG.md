# Changelog

All notable changes to this installer distribution are documented in this file.

The format is based on Keep a Changelog style and adapted for installer releases.

> Current publication status: `v1.0.0` (2026-03-24) is the first and currently only published app release.

## [v1.0.0] - 2026-03-24

### Release Status
- First published release
- Currently the only published release

### Installers
- `setup.exe`
- `YT-M Downloader.msi`

### Added
- Initial public Windows installer distribution for Ultimate YT Downloader (GUI).
- GUI workflow for URL input, media type selection (`single`, `playlist`, `audio`), and profile selection (`compatible`, `best`).
- Built-in **Run Setup** action to initialize required runtime tools.
- Saved defaults support for repeat usage.
- Optional subtitle download toggle.
- Dark mode option.
- Output panel controls (`Copy Output`, `Clear Output`) and runtime status display.
